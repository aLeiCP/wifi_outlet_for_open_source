## 前言
该代码仓主要存放的是红旭无线发起的基于ESP32的WIFI插座开源项目的相关资料.借Github这块宝地,很荣幸能跟五湖四海的朋友一起共事,协作完成此次的开源项目.在这里我代表红旭无线开发团队谢谢具有同样兴趣爱好的朋友的支持.自2018年07月18日发起至今,随着参与人数的增加,现今有了如下的进展,接下来将由我给大家讲解.
